Formas de agregar estilos a nuestra página web

Estilos en Línea

Estos estilos se agregan directaente en el elemento html, mediante el atributo style

Este párrafo tiene estilos en línea.

Estilos Internos

Estos estilos se agregan en el mismo archivo html, dentro de la etiqueta <head>

Estilos Externos

Estos estilos son los que se recomiendan, ya que ofrecen modularidad, fácil mantenimiento, orden, eficiencia, entre otras ventajas, para agregar una hoja de estilos externa, dentro de la etiqueta <head> se va a utilizar la etiqueta <link>
